# Darul Uloom Website Package

Files:
- index.html -> main website
- admin.html -> simple content editor
- content.js -> all editable content
- style.css -> colors and design
- script.js -> dynamic rendering
- robots.txt -> crawler instructions
- sitemap.xml -> sitemap template
- manifest.json -> installable web app basics
- assets/logo.jpg -> current uploaded logo

Before going live:
1. Replace https://example.com in content.js, robots.txt and sitemap.xml with your real website URL.
2. Put your real mosque/building images in assets/ and update content.js.
3. Upload the full folder to Netlify, Vercel, GitHub Pages, or your hosting panel.
